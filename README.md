# Pelican

YouTube web server with real-time queue.

[![Build Status](https://secure.travis-ci.org/fragphace/pelican.png?branch=master)](http://travis-ci.org/fragphace/pelican)

## Installation

	npm install -g pelican

## Start

Pelican starts as a daemon:

	pelican start

To stop the server:

	pelican stop

## Tests

	npm test
